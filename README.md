# OncoShape3D

Versione con navigazione cliccabile e logo OS³D.

## Funzioni

- Riquadri Home / Analisi STL / Metodo / Disclaimer / Contatti cliccabili
- Logo OS³D
- Upload multiplo STL
- Visualizzatore 3D interattivo
- Risultati verticali
- Export Excel/CSV

## Avvio

```bash
pip install -r requirements.txt
streamlit run app.py
```
